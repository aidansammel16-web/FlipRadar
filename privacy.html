Privacy Policy for FlipRadar

FlipRadar is designed to monitor user-provided Facebook Marketplace searches and notify users about new listings and price changes.

Data Storage

FlipRadar stores user settings, Marketplace search URLs, seen listing identifiers, and configuration data locally on the user's device using the browser's storage system. This data is not transmitted to the developer.

Data Collection

FlipRadar does not collect or store any personal data on external servers. All data is processed locally within the user's browser. Data is only transmitted externally if the user explicitly enables optional Telegram notifications.

Use of Website Content

The extension accesses Facebook Marketplace pages solely to extract listing information such as title, price, and URL in order to evaluate matches based on user-defined criteria.

Telegram Notifications (Optional)

If the user enables Telegram alerts and provides their own bot credentials, listing information (such as title, price, and URL) is sent to Telegram using the Telegram Bot API. This functionality is entirely optional and initiated by the user. No Telegram credentials or message data are stored or accessed by the developer.

Data Usage

All data is used exclusively for the core functionality of monitoring Marketplace listings and notifying the user. FlipRadar does not use analytics, tracking technologies, or advertising systems.

Security

All data processing occurs locally within the user's browser unless the user explicitly enables Telegram notifications.

Changes

This privacy policy may be updated in the future. Continued use of the extension constitutes acceptance of any updates.
